Module:3

  1. Github is a platform where developers can safely store ,modify,share their code.


  2. To creat a new repository,
    • click on new
    • Give your repository a name.
    • open a folder in vs code where we will write our code.
    • Run the 6 line command in vs code terminal.


  3. After making a little change in our code,we need to Run 3 line command to update it in our git repository.
    • Add - git add.
    • Commit- git commit -m "commit text"
    • Push- git push


  4. To host or publish,go to settings -> then github pages --> and generate the link .
